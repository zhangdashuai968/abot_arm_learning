**阶段**：Phase 1
**预计耗时**：8-12 小时（分 4-6 次，每次 2 小时）
**前置**：无（可与 E01/E02 并行；建议在自己虚拟机上做，不占整车）

- E03 ROS 乌龟仿真实验

- E06 ROS 动作使用实验

## 目标

把 ROS1 通信机制（topic / service / action）+ 调试工具（rviz / rqt）+ 串口绑定吃透，能徒手写一个完整的"发布-订阅 + 服务 + action"小项目。

## 阅读策略

- **第一遍速读**：每个实验讲义 5 分钟过一遍，画思维导图
- **第二遍精读 + 仿做**：每个实验照着跑一遍命令，截图存档
- **第三遍输出**：合上讲义，凭记忆+终端历史写笔记

## 7 个实验的关键产出

### E03 乌龟仿真（topic 入门）
- 跑 `turtlesim` + 用 `rostopic pub` 控制乌龟
- 写一个发布者节点让乌龟画圆
- **关键概念**：topic / publisher / subscriber

### E04 消息使用
- 自定义 `.msg` 文件
- **关键概念**：std_msgs vs 自定义 msg

### E05 服务通信
- 写一个加法服务
- **关键概念**：service vs topic 的差异（同步 vs 异步）

### E06 动作使用
- 写一个倒计时 action（带 feedback / result）
- **关键概念**：action = service + 进度反馈

### E07 rviz 工具
- 在 rviz 里可视化 tf / pointcloud / image
- **关键概念**：fixed frame、display 类型

### E08 rqt 工具
- `rqt_graph` / `rqt_plot` / `rqt_console` / `rqt_reconfigure`
- **关键概念**：调试用什么工具

### E09 串口通信及绑定
- 用 `udev` 规则把串口绑定到固定名（如 `/dev/abot_imu`）
- **关键概念**：为什么要绑定（重启后 ttyUSB0/1 顺序会乱）
