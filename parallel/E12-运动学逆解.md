# E12 · 机械臂运动学逆解

**阶段**：Phase 2
**预计耗时**：3-4 小时（逆解比正解难一档）
**前置**：E11 已完成
**对应官方实验讲义**：`~/桌面/abot开源机器人/v01.01.01.0000/实验12_运动学逆解实验.md`

## 目标

理解逆解为什么比正解难，能调用 MoveIt 的 IK 求解器拿到关节角，并看出"奇异点"现象。

## 关键问题（执行前先尝试用直觉回答）

1. 逆解为什么有"多解"？6 自由度机械臂典型有几组解？
2. 解析解（封闭式）与数值解（迭代）哪种快？哪种稳？工业上用哪种多？
3. 奇异点是什么？为什么靠近奇异点 IK 容易爆炸？最常见的奇异姿态长啥样？
4. 工作空间外的位姿，IK 会返回什么？（None? 报错? 给一个最近的解？）
5. 同一个目标位姿，给不同初值（seed）会得到不同解吗？

## 执行步骤

### 1. 用 MoveIt 玩 IK
```bash
roslaunch <arm_pkg>_moveit_config demo.launch    # 或厂家给的 moveit launch
```
- 在 rviz 拖动末端到几个位置，看 MoveIt 给出的关节角
- 记录 1-2 组（位姿，关节角）对照

### 2. 手算一组逆解（选简单姿态）
- 针对一个"末端朝下、纯平移到 (x, y, z)"的简单目标
- 用几何法手推 1-3 关节，再用解耦法推 4-6 关节
- 记录在 `notes/正逆解推导.md` 第二节

### 3. 奇异点观察
- 让机械臂走到一个肘关节伸直的姿态
- 看 MoveIt 是否报 "near singularity"
- 改几个微小目标看关节角是否大跳

### 4. 写逆解调用脚本
```python
# 用 moveit_commander 调用
group.set_pose_target([x, y, z, qx, qy, qz, qw])
plan = group.plan()
```
对比 E11 正解：取 IK 输出的关节角 → 跑 FK → 看是否回到目标位姿。
