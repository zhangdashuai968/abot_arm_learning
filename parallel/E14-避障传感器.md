## 目标

搞清楚整车上"避障传感器"具体是哪些（超声波 / 红外 / 激光雷达 / 防撞栏 / 深度相机），各自的话题、量程、盲区，画出一张避障传感器布局图。

## 关键问题

1. 这台车的"避障传感器"和"激光雷达 A3M12"是同一个吗？还是另有低成本超声/红外？
2. 超声波（如有）的量程一般 0.02-4 m，盲区在哪？多个超声会不会互相干扰？
3. 深度相机 <!-- 2026-05-03 lsusb实测修正 --> 深度图能不能用作避障？相比激光雷达优劣？
4. 防撞栏（机械式）触发后通过什么方式上报到 ROS？
5. 多传感器避障如何融合？由 navigation stack 的 costmap 统一处理？

## 执行步骤

### 1. 找传感器
- 在车上找：超声波 / 红外发射头 / 防撞栏开关
- 看 `rostopic list | grep -i -E "range|sonar|ir|bumper|scan"`

### 2. 读数据
对每个找到的传感器：
```bash
rostopic info /<topic>     # 消息类型
rostopic hz /<topic>       # 频率
rostopic echo -n3 /<topic> # 看几帧
```

### 3. 测量量程与精度
- 在传感器前 0.1 / 0.5 / 1.0 / 2.0 m 处放一个目标
- 记录读数 vs 真实距离
- 算精度（绝对误差 + 相对误差）

### 4. 画布局图
- 俯视图标 4 个超声 / 防撞栏 / 雷达位置
- 画出每个传感器的可视范围（扇形/圆锥）
