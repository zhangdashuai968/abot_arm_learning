# 速查 · abot 机械臂车启动链路（雏形）

> 本文件是占位骨架，待 E02 实物验证后填充真实命令
> 参考 amov_uav_learning/cheatsheets/启动链路.md 的结构

## N 步标准启动（已验证）

```bash
# ① 加载环境变量
source ~/.bashrc

# ② 进入工作空间
cd ~/catkin_ws

# ③ 编译项目
catkin_make

# ④ 设置环境（让系统找到刚编译的包）
source devel/setup.bash

# ⑤ 启动整机（bringup 总入口）
roslaunch abot bringup.launch
```

### 一键版（合并所有步骤）

```bash
source ~/.bashrc && cd ~/catkin_ws && catkin_make && source devel/setup.bash && roslaunch abot bringup.launch
```

## 一键关键检查（待验证）

```bash
ls /dev/tty*                                 # 串口设备
lsusb                                        # USB 设备
ip addr | grep 192                           # 网线 IP
rostopic hz /scan                            # 雷达发布率
rostopic hz /camera/color/image_raw          # 相机发布率
rostopic hz /imu/data                        # IMU 发布率
rostopic echo -n1 /joint_states              # 机械臂关节角
```

## 关停顺序

反向 Ctrl+C，最后断电。机械臂记得回 home 再断电（避免下次开机奇异点）。

## 常用 launch（待 E02 后填真实清单）

| Launch | 用途 |
|---|---|
| `<bringup>/main.launch` | 整机起 |
| `<bringup>/base_only.launch` | 仅底盘 |
| `<bringup>/arm_only.launch` | 仅机械臂 |
| `<bringup>/sensors_only.launch` | 仅传感器 |

## 节点角色（一句话版，待补充）

- `roscore` 在工控机上跑（待确认 Master URI）
- 底盘走 `/cmd_vel` → 底盘控制器 → 4 个轮电机
- 机械臂走 `/joint_command` 或 MoveIt 接口 → ROCR6 driver → 关节
- 深度相机（Orbbec）走 USB 3.0；RPLidar 走串口；IMU 走串口
