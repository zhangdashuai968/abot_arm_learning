# 速查 · abot 机械臂车启动链路

> 最后更新：2026-05-26
> 来源：small-car 真机 + E02 实物验证

---

## N 步标准启动

```bash
# ① 加载环境变量
source ~/.bashrc

# ② 进入工作空间
cd ~/catkin_ws

# ③ 编译项目
catkin_make

# ④ 设置环境（让系统找到刚编译的包）
source devel/setup.bash

# ⑤ 启动整机（bringup 总入口）
roslaunch abot bringup.launch
```

### 一键版

```bash
source ~/.bashrc && cd ~/catkin_ws && catkin_make && source devel/setup.bash && roslaunch abot bringup.launch
```

---

## 常用启动组合

```bash
# 完整导航
roslaunch abot bringup.launch && roslaunch abot navigate.launch

# 仅建图
roslaunch abot bringup.launch && roslaunch abot slam/cartographer_slam.launch

# 仅底盘（调试用）
roslaunch abot bringup.launch
```

## 关键检查

```bash
ls /dev/tty*                                 # 串口设备
lsusb                                        # USB 设备
ip addr | grep 192                           # 网线 IP
rostopic hz /scan                            # 雷达发布率
rostopic hz /camera/color/image_raw          # 相机发布率
rostopic hz /imu/data                        # IMU 发布率
rostopic echo -n1 /joint_states              # 机械臂关节角
```

---

## 关停顺序

反向 Ctrl+C，最后断电。机械臂回 home 再断电（避免下次开机奇异点）。

---

## 节点角色

- `roscore` 在 Jetson Nano 上跑（Master URI: `http://192.168.36.46:11311`）
- 底盘走 `/cmd_vel` → 底盘控制器 → 4 个轮电机
- 机械臂走 MoveIt 接口 → ROCR6 driver → 6 个关节
- 深度相机（Astra）走 USB 3.0；RPLidar 走串口；IMU 走串口
